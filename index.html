DSharp Model Viewer

Tip: Select your DSharp project root folder. The viewer looks for model.json files or classes/*.json per model. References can be class IDs or paths like "Model/SubModel/ClassName".

Assumptions: Each model is a directory; optional model.json with {name, classes, submodels}. Classes may also be in classes/*.json. A class belongs to one model but may be referenced elsewhere; such visiting classes are shown with dashed borders.
